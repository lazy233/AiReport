**通用字段**

这部分模板几乎都能共用。


|              |         |                       |
| ------------ | ------- | --------------------- |
| 字段名          | 显示名     | 说明                    |
| student_name | 学生姓名    | 姓名                    |
| grade_intake | 年级/入学时间 | 如 `2025级` / `2025/09` |
| school       | 学校      | 学校名称                  |
| major        | 专业      | 专业名称                  |
| planner      | 规划老师    | 规划/顾问老师               |
| advisor      | 教服老师    | 教服/BD/对接人             |
| product      | 产品      | 服务产品名                 |
| term         | 学期/阶段   | 如 `大一上`、`2025秋`       |


**选课/规划模板**

目录建议压缩成：

1. 学生信息
2. 需求分析
3. 毕业要求
4. 选课规则
5. 推荐课程
6. 学期规划
7. 长期规划
  1. 

对应字段：


|                     |         |                |
| ------------------- | ------- | -------------- |
| 字段名                 | 显示名     | 说明             |
| strength_subjects   | 擅长科目    | 如英语/数学         |
| scores              | 语言/国际成绩 | 如雅思/AP/A-Level |
| learning_good       | 擅长学习形式  | 如考试/实验         |
| learning_weak       | 不擅长学习形式 | 如论文/演讲         |
| interests           | 兴趣方向    | 感兴趣科目/领域       |
| study_goal          | 升学意向    | 硕士/转专业等        |
| career_goal         | 就业意向    | 行业/岗位方向        |
| long_goal           | 长远目标    | 长期发展目标         |
| degree              | 学位      | 学位名称           |
| duration            | 学制      | 几年制            |
| credits             | 学分要求    | 总学分/分项学分       |
| course_rule         | 课程要求    | 必修/选修要求        |
| gpa_rule            | GPA要求   | GPA或等级要求       |
| selection_rule      | 选课规则    | 先修/冲突/时间要求     |
| recommended_courses | 推荐课程    | 推荐课程清单         |
| course_notes        | 课程说明    | 对重点课程的说明       |
| prep_courses        | 预习课程    | 开学前建议          |
| tutoring_courses    | 同步辅导课程  | 建议跟课课程         |
| term_plan           | 学期规划    | 本学期重点安排        |
| future_plan         | 后续规划    | 后续学期安排         |


**学期总结/结单模板**

目录建议压缩成：

1. 基本信息
2. 辅导执行
3. 成绩情况
4. 服务总结
5. 下阶段规划
6. 结单信息
  1. 

对应字段：


|                  |       |              |
| ---------------- | ----- | ------------ |
| 字段名              | 显示名   | 说明           |
| student_summary  | 学生情况  | 学生整体情况概述     |
| school_ddl       | 校方DDL | 重要截止时间       |
| first_class_time | 首课时间  | 首节课日期        |
| first_class_note | 首课记录  | 首节课观察        |
| summer_work      | 暑期辅导  | 暑期计划/成果      |
| term_work        | 学期辅导  | 本学期计划/成果     |
| recorded_courses | 录播完成  | 已完成录播课       |
| total_hours      | 总课时   | 总课时数         |
| used_hours       | 已用课时  | 已消耗课时        |
| left_hours       | 剩余课时  | 剩余课时         |
| grades           | 成绩明细  | 各科成绩         |
| gpa              | GPA   | 当前GPA        |
| target_gpa       | 目标GPA | 阶段目标         |
| final_score      | 最终成绩  | 课程最终结果       |
| services         | 服务内容  | 邮箱/反馈/咨询等    |
| service_count    | 服务次数  | 实际服务次数       |
| class_count      | 课程次数  | 上课次数         |
| total_duration   | 总时长   | 全部课程时长       |
| avg_duration     | 平均课时  | 单节平均时长       |
| communication    | 沟通频次  | Classin/群内沟通 |
| next_goal        | 下阶段目标 | 下学期目标        |
| risk_courses     | 风险科目  | 需重点关注课程      |
| suggestions      | 下阶段建议 | 后续建议         |
| remarks          | 备注    | 其他备注         |


**不标准化字段**

这部分建议全部做成`人工填写区`，不要结构化太细。


|                 |              |
| --------------- | ------------ |
| 字段类型            | 用途           |
| analysis_text   | 学员当前情况分析     |
| feedback_text   | 老师/学生反馈      |
| summary_text    | 阶段性学生总结      |
| suggestion_text | 个性化建议        |
| special_case    | 特殊情况说明（如心理等） |
